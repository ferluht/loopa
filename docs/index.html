LOOPA

A portable open-source sound computer based on raspberry pi zero

Portable

60x140mm height and witdth, ~12mm thickness, with built-in battery

Powerful

1GHz + 512Mb RAM RPi zero, SGTL5000 audio codec, line in, line out, microphone, usb host, wifi, bluetooth, 128x32 OLED display

Customizable

All the software and hardware are going to be open-source, allowing people to add instruments, design new effects, or even completely change LOOPA functionality
